## 概览
本规范定义了符合高校教学标准的幻灯片结构要求。

## 章-节-知识点三级结构

### 结构示例
```
第一章 数据结构基础
├── 1.1 数组
│   ├── 1.1.1 数组的定义与特点
│   ├── 1.1.2 数组的内存结构
│   └── 1.1.3 数组的应用
├── 1.2 链表
│   ├── 1.2.1 单链表
│   ├── 1.2.2 双向链表
│   └── 1.2.3 循环链表
└── 1.3 栈和队列
    ├── 1.3.1 栈的定义与操作
    ├── 1.3.2 队列的定义与操作
    └── 1.3.3 栈和队列的比较
```

### JSON配置示例
```json
[
  {
    "title": "第一章 数据结构基础",
    "content": "章节介绍",
    "fragments": ["数组", "链表", "栈和队列"]
  },
  {
    "title": "1.1 数组",
    "subtitle": "1.1.1 数组的定义与特点",
    "content": "数组的定义...",
    "formulas": ["$$a[i] = address + i \\times size$$"],
    "fragments": ["特点1", "特点2", "特点3"]
  }
]
```

### 标题规范
- 章标题：使用"第一章"、"第二章"格式
- 节标题：使用"1.1"、"2.3"格式
- 知识点标题：使用"1.1.1"、"2.3.1"格式

## 幻灯片类型

### 1. 章节封面幻灯片
用于章节标题和章节概述。

**配置示例：**
```json
{
  "title": "第一章 数据结构基础",
  "content": "本章介绍数据结构的基本概念、数组和链表的核心原理...",
  "fragments": ["1.1 数组", "1.2 链表", "1.3 栈和队列"],
  "thought_question": "为什么需要数据结构？"
}
```

### 2. 知识点幻灯片
用于讲解具体知识点。

**配置示例：**
```json
{
  "title": "1.1.1 数组的定义",
  "subtitle": "核心概念",
  "content": "数组是相同类型数据的有序集合...",
  "fragments": [
    "特点1：连续内存存储",
    "特点2：随机访问",
    "特点3：固定大小"
  ],
  "formulas": ["$$a[i] = \\text{base} + i \\times \\text{element_size}$$"],
  "thought_question": "数组有哪些优点和局限性？"
}
```

### 3. 原理说明幻灯片
用于阐述理论原理和公式推导。

**配置示例：```json
{
  "title": "冒泡排序原理",
  "subtitle": "1.2.1 算法原理",
  "content": "冒泡排序通过相邻元素比较和交换实现排序",
  "formulas": [
    "$$O(n^2)$$",
    "$$T(n) = \\sum_{i=1}^{n-1} (n-i) = \\frac{n(n-1)}{2} = O(n^2)$$"
  ],
  "fragments": [
    "第一轮：最大元素冒泡到末尾",
    "第二轮：次大元素冒泡到倒数第二",
    "重复直到排序完成"
  ]
}
```

## 内容密度控制

### 单张幻灯片最大容量
- 标题：最多2行
- 正文：最多5个要点
- 公式：最多3个公式
- 动画/图表：仅限1个
- 思考题：1个

### 内容过多时的处理
- 使用垂直幻灯片拆分
- 使用fragments分步展示
- 使用链接到详细参考文档

### 代码示例
**正确：**
```json
{
  "title": "冒泡排序",
  "fragments": ["步骤1", "步骤2", "步骤3", "步骤4"],
  "thought_question": "时间复杂度是多少？"
}
```

**错误（过多）：**
```json
{
  "title": "冒泡排序",
  "fragments": ["步骤1", "步骤2", "步骤3", "步骤4", "步骤5", "步骤6", "步骤7", "步骤8", "步骤9", "步骤10"],
  "formulas": ["公式1", "公式2", "公式3", "公式4", "公式5"],
  "thought_question": "思考题"
}
```

## 思考题设计

### 思考题类型
1. **回顾类问题**：回顾和巩固核心概念
2. **分析类问题**：分析原理和关系
3. **综合类问题**：联系实际应用
4. **延伸类问题**：引导深入思考

### 设计原则
- 简洁明了，易于理解
- 与本张幻灯片内容相关
- 引导思考，不只是记忆
- 有一定难度，但可回答

### 示例
```
- 回顾类："冒泡排序的时间复杂度是多少？为什么？"
- 分析类："在什么情况下链表比数组更优？"
- 综合类："如何根据实际需求选择合适的数据结构？"
- 延伸类："冒泡排序有哪些改进方法？"
```

## 公式使用规范

### LaTeX语法
- 行内公式：使用 `$...$`
- 块级公式：使用 `$$...$$`

### 常用数学符号
```latex
求和: $$\sum$$
积分: $$\int$$
分数: $$\frac{a}{b}$$
上标: $$x^n$$
下标: $$x_i$$
希腊字母: $$\alpha, \beta, \gamma$$
```

### 公式示例
```json
"formulas": [
  "$$O(n^2)$$",
  "$$T(n) = \\sum_{i=1}^{n} f(i)$$",
  "$$\\frac{d}{dx}f(x) = \\lim_{h\\to 0}\\\\frac{f(x+h)-f(x)}{h}$$"
]
```

## 代码展示规范

### 代码格式
- 使用HTML `<pre><code>` 标签
- 可以使用语法高亮

### 示例
```json
{
  "content": "<pre><code class=\"python\">def bubble_sort(arr):\n    for i in range(len(arr)-1):\n        for j in range(len(arr)-1-i):\n            if arr[j] > arr[j+1]:\n                arr[j], arr[j+1] = arr[j+1], arr[j]\n    return arr</code></pre>"
}
```

## 注意事项

1. **结构清晰**：标题层级明确
2. **内容精炼**：避免冗长描述
3. **视觉平衡**：文字、公式、动画/图表合理搭配
4. **教育价值**：每个元素都要有明确的教学目的
5. **响应式考虑**：确保不同设备上可读
6. **学术规范**：术语准确、引用明确
